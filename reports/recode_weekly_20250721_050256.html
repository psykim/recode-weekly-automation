최근 연구 결과들은 가소제 디-2-에틸헥실 프탈레이트(DEHP) 노출이 신경행동학적 변화, 인지기능 장애, 신경퇴행을 포함한 신경학적 결과를 초래한다는 것을 강조했습니다. 소포체(ER) 스트레스는 세포 사멸을 유도하여 세포 반응을 변화시키는 데 있어 독특한 역할을 합니다. 기존 문헌을 바탕으로, 저희는 DEHP의 총체적인 생화학적 및 신경형태학적 결과가 신경 미세환경에서 소포체 스트레스 증가와 관련이 있는지 여부를 이해하는 데 주된 관심을 두었습니다. 또한, DEHP로 유발된 소포체 스트레스에 대한 제브라피시 뇌의 NAC(N-acetyl cysteine)의 신경보호 효능을 규명하고자 했습니다. 저희의 핵심 연구 결과는 DEHP가 광범위한 신경독성 물질로서 산화 스트레스 증가와 칼슘 항상성 조절 장애를 유발하며, 이는 제브라피시 뇌에서 ATF4 의존적 GRP78의 상향조절을 통해 소포체 스트레스 매개 신경퇴행에 기여한다는 주장을 강력하게 뒷받침합니다. 그러나 NAC는 DEHP로 인한 칼슘 조절 장애(F4,20 = 11.97, p < 0.05; n = 6/그룹)와 ATF4로 인한 ER 스트레스(F4,8 = 11.05; p < 0.05; n = 6/그룹)를 GRP78(F4,8 = 31.83; p < 0.05; n = 6/그룹)을 억제함으로써 역전시켰습니다. NAC는 또한 내인성 BDNF(F₄,₈ = 13.93, p < 0.05; n = 6/그룹) 및 NeuN(F₄,₈ = 56.95, p < 0.05; n = 6/그룹) 발현의 상향 조절과 다니오 뇌의 주변실 회색 지대(PGZ)에서 CC3-매개(F₄,₈ = 22.03, p < 0.05; n = 6/그룹) 신경 변성의 억제를 통해 신경 보호를 촉진했습니다. 이러한 관찰 결과를 요약하자면, 저희 연구는 NAC 공동 보충과 다니오에서 DEHP-유도 ER 스트레스 및 신경 병리학의 회복 사이에 강력한 상관 관계가 있음을 확인했지만, DEHP에 대한 NAC-매개 잠재적 치료제 개발을 보장하기 위해서는 추가 검증이 필요합니다.

베타 아밀로이드(Aβ) 플라크와 타우 엉킴은 알츠하이머병(AD)의 병리학적 특징입니다. 그러나 부검 연구에 따르면 대부분의 노인은 뇌혈관 질환(CVD) 지표를 동반하는 것으로 나타났습니다. A+T+ 성인은 A-T- 성인에 비해 사망 전 약 10년 동안 모든 결과에서 유의미하게 빠른 인지 저하를 보였습니다(d=0.34-0.46). 최종 방문 치매 진단율이 낮은 상황에서 A+T+ 성인의 인지 저하가 더 빠른 것은 사후 A+T+가 사망 전 가파른 인지 저하 경로와 관련이 있지만, 치매 진행이 불가피한 것은 아님을 시사합니다. (핵심 내용 요약)
